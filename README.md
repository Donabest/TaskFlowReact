TaskFlow – Modern React Task Manager

TaskFlow is a modern task management application built to practice scalable React architecture, server state management, and clean UI interactions.
The project focuses on separating concerns between API services, custom hooks, and UI components while using React Query for server state synchronization.

Live Demo
taskflowreact.vercel.app

Features
	•	Create tasks
	•	Edit tasks
	•	Delete single tasks
	•	Delete all tasks with confirmation modal
	•	Real-time UI updates with React Query
	•	Search tasks instantly while typing
	•	Toast notifications for feedback
	•	Modal confirmation before destructive actions
	•	Responsive UI layout

Tech Stack

Frontend
	•	React
	•	React Query
	•	React Hook Form

Backend / Database
	•	Supabase

UI / UX
	•	React Hot Toast
	•	Framer Motion (for animations)
